# eng
AI现在太火爆了，不学，感觉就被淘汰了。正好，有个小需求，我就顺带试试水，看看AI现在到底进化到什么程度......

闲来无事，用AI写个小网站：少儿学习英语+答题试小游戏
使用到的AI工具：豆包+千问+Trae，均为免费软件
豆包：用来获取数据源，因为它的响应速度是最快的，但基本上写不了代码
千问：它帮我写最初的代码及样式设计，但每次交互时间太长，且不能直接修改我的代码，只能在网页复制来复制去的，速度也非常慢
Trae：最初的代码由千问生成后，Trae 帮忙优化样式、修改BUG、开发新功能、整合从豆包获取的数据 ，主要偏向逻辑层处理，速度挺快

开发共计耗时：2天半
所有代码均由AI生成，未手动写一行代码
人为分析过一次：是某个页面 Trae 改了 5次没有效果，我帮他分析，告知AI，解决了问题

总结：AI发展的速度真的是太快了，这次我一行代码没写，整个网站都能做好。在使用期间，AI真的很智能，根本不需要特定的指令，就和生活中，和某个人正常聊天一样，使用成本超低。
预判：5年后，80%的程序员都会失业。留下10%超高级的程序员，用于对AI的开发以迭代，剩下次高级的程序员的工作方式也会发生改变，更多的是驱动AI写代码，自己不需要再写代码了

后怕的是：我使用的工具是免费的，且都是半年前的东西，和收费的比起来效率挺低。另外，现在都是ai agent模式，也就是 openclaw 模式，效率还会进一步提升
